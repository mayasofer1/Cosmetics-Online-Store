# React + Node.js E-commerce Cosmetics Store

This project is a full-stack e-commerce application for a cosmetics store, built using React for the client side and Node.js with MongoDB for the server side. 

## Client Side

The client side of the application is structured as follows:

- **Components**: Contains reusable components such as Navbar, ProductList, ProductCard, Cart, and OrderPage.
- **Pages**: Contains main pages like Home, CartPage, and OrderPage.
- **Assets**: Contains images and other static assets.

### Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd my-cosmetics-store/Client
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the development server**:
   ```
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the application.

### Features

- **Navigation Bar**: A responsive navigation bar that links to the home page and shopping cart.
- **Home Page**: Displays a list of products with images, descriptions, and prices.
- **Shopping Cart**: Allows users to add products, view their cart, and modify quantities.
- **Order Processing**: Users can review their cart and enter shipping information to complete their order.

### Technologies Used

- **Frontend**: React, Vite, CSS
- **Backend**: Node.js, Express, MongoDB
